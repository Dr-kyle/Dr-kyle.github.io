Documents API

  • 指定 id 将会覆盖或者新建
  • 没有 id 将会自动生成
  • 文档最多可包含 64 个键值对
  • 每个请求最大为100个文档，每个文档必须小于 100kb
  • 索引请求不能超过 10mb
  • 字段名称不能是 external_id, engine_id, highlight, or, and, not, any, all, none
